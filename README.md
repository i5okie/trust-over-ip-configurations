# Trust Over IP Configurations

This repository contains the workflow for building and deploying vc-authn-oidc with custom templates. A workflow for regularly re-installing Traction sandbox. And proof configurations for vc-authn-oidc in all environments.

Configurations for deploying other services with ArgoCD have been migrated to a private GitOps repository.
